# 更新日志

## v2.0.0 (2024-12-14)

### 新增
- GUI 图形界面版本（Python + tkinter）
- 14 项修复功能，可自由选择
- 实时进度条和日志显示
- 自动请求管理员权限（UAC 提权）
- Win7 SP1 ~ Win11 全兼容（内置 VC++ 运行时库）

### 改进
- 从原版 BAT 脚本升级为 Python GUI
- 修复项目可单独选择执行
- 增加执行状态提示和日志记录
- 网络类型自动设置为专用网络

### 修复
- 修复 0x80004005 SMB 客户端不安全身份验证问题
- 修复 0x0000011b 打印机共享错误
- 修复 0x80070043 网络名称无效问题
- 修复 709 错误（RPC 加密级别）
